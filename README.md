# Fantasy Baseball Decision Engine

An AI-powered fantasy baseball assistant that helps optimize daily lineup decisions using live sportsbook odds, season statistics, roster construction, and league-specific scoring rules.

## Features

- Live MLB sportsbook odds
- Fantasy point projections
- Daily lineup optimization
- Waiver wire recommendations
- Add/drop analysis
- Risk-adjusted decision engine
- Position-specific replacement value
- Streaming hitter strategy
- Multi-add optimization
- Long-term roster protection
